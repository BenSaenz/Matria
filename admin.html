Gestión local HTML

Colecciones, productos y cupones

Este panel funciona en GitHub + Cloudflare Pages sin backend. Los cambios se guardan en tu navegador mediante localStorage y luego se exportan como JSON para reemplazar los archivos de assets/data/ en tu repositorio.

Las imágenes y videos deben guardarse en carpetas físicas dentro de assets/img/. Desde aquí administras su ruta relativa y metadatos: alt, title y caption.
